# Node.js를 이용한 CRUD 게시판 만들기
<br>

## ⏳ 개발기간

약 1개월 (2023.06.25 ~ 2023.07.31)

<br/>

## 📖 프로젝트 설명

Node.js - Express를 이용한 간단한 CRUD 게시판입니다. <br>
게시판에 게시된 게시물들을 확인할 수 있으며, 회원가입 및 로그인 이후 게시글 및 댓글을 작성할 수 있습니다. <br>
게시글은 게시글 작성자만이 게시글을 수정 및 삭제할 수 있습니다.
